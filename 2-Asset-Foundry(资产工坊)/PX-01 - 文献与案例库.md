# 信息炼金台

> 本库只存放经过深度处理、符合严格标准的“知识晶体”，拒绝原始信息和偏见。

## 晶体库
```dataview
TABLE 来源, 证据强度, 核心结论
FROM "5-Knowledge-Garden"
WHERE contains(tags, "#knowledge-crystal")
SORT 证据强度 DESC
```

## ⚗️ 信息炼金流程
所有信息必须经过以下流程才能入库：
1.  **溯源**：信息来自何处？是一手信源（论文、财报、数据）还是二手转述？
2.  **批判**：作者的核心立场是什么？他/她可能有什么偏见？
3.  **交叉验证**：是否有其他独立信源支持这个结论？
4.  **提炼**：用**自己的话**总结核心观点或数据，并注明来源。
5.  **标签化**：打上 `#knowledge-crystal` 标签。

## 添加新晶体
**模板**：
```markdown
# [简短结论作为标题]
-   **来源**：[[原文链接]] 
-   **证据强度**： (1-5⭐) 
-   **核心事实/数据**： 
-   **我的解读/批判**：
-   **标签**： `#knowledge-crystal` `#来源-[机构名]` `#topic-[主题]`
```
