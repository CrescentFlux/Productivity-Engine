# 人脉账本：价值驱动连接

> 本账本记录基于价值交换的弱连接建设。原则：**不求贵人，只寻伙伴；不谈索取，先谈贡献。**

## 🧠 我的价值清单
> 在连接他人前，先清晰自己的价值。我可以免费提供什么？
-   **技能价值**：我能帮人解决GitHub/Obisidian/知识管理的问题。
-   **信息价值**：我擅长搜集和整理某一领域的优质信息（金融、物理）。
-   **反馈价值**：我能为人提供真诚、详细的内容反馈。

## 🔍 目标连接对象
> 我希望认识什么样的人？（必须具体）
-   [ ] 独立开发者 / 数字游民 (学习变现路径)
-   [ ] 科技、金融领域的英文译者 (学习语言和专业)
-   [ ] 同样在构建个人知识系统的同道中人 (互相激励学习)

## ✅ 执行清单 (Zero-Cost Actions)
> 无需花钱，只需花时间的行动。

### 1. 价值输出式连接 (最佳方式)
-   [ ] **行动**：在目标对象(如某译者)的社交媒体下，写一段**极其详细、真诚的评论**，说明他的内容具体好在哪里，以及对我产生的启发。
-   **记录**：
    | 日期 | 对象 | 平台 | 我的动作 (提供的价值) | 结果/下一步 |
    | :--- | :--- | :--- | :--- | :--- |
    | 2024-05-28 | @某某译者 | Twitter | 对其一篇译文写了300字的长评，分析其翻译妙处 | 对方回复感谢，并关注了我 |

### 2. 提问式连接
-   [ ] **行动**：经过深入研究后，提出一个**高质量的、对方专业领域内**的问题。
-   **记录**：
    | 日期 | 对象 | 平台 | 我的动作 (提供的价值) | 结果/下一步 |
    | :--- | :--- | :--- | :--- | :--- |
    | 2024-05-29 | @某开发者 | GitHub | 为其开源项目发现了一个文档中的小错误并提出修改建议 | 对方Merge了我的PR |

### 3. 分享式连接
-   [ ] **行动**：当我创作了高质量内容（如你的金融科普），**定向分享**给可能会感兴趣的朋友（“想起你之前聊过这个话题，我的这篇笔记你可能会有启发”）。
-   **记录**：(同上)
